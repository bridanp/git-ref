Stash Commands - temporary storage for modifications

git stash [save] []

Store all local modifications to a stash. Save and the msg parameters are optional.

git stash apply []

Brings in changes from the specified stash to the work area. Ommit the and it pulls the default stash.

git stash pop

Apply the changes from the last pop and delete that stash. Can specify a stash at the end.

git stash list

List all of the stashes currently available.

git stash show []

Outputs the difference in the stash and the parent. can be left off and the system will show the latest stash.

git stash drop []

Remove one stash from the list. If the is left off, the command drops the latest version.

git stash clear

Removes all stashes.

git stash branch []

Create and checkout the branch named using the listed. If is left off of the command, the latest is used.
